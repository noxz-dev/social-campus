# Social Network

- Built on Node version 14.16
- Typescript
- Vue 3
- TypeORM
- GraphQL
- Postgres
- Docker
- Minio

## Startup

- run npm install in network_backend

- run npm install in network_frontend

- add .env file to the root folder for container configuration (.env-sample is given)

- run docker-compose up -d
